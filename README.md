# Service Operations & SLA Analytics Dashboard

## Project Overview
This project focuses on analyzing technical support operations data to evaluate service efficiency, SLA compliance, customer satisfaction, and agent performance. The analysis was conducted using SQL, Excel, and Power BI to identify operational bottlenecks and derive actionable business insights.

The dashboard was built using a dataset containing over 2,300 technical support tickets across different support channels, issue categories, countries, and support levels.

---

## Business Objectives
- Monitor SLA compliance across support operations
- Identify key drivers of SLA violations
- Analyze agent performance and handling efficiency
- Evaluate customer satisfaction trends
- Understand ticket distribution across channels and priorities
- Identify operational bottlenecks requiring improvement

---

## Tools & Technologies Used
- SQL
- Excel / Google Sheets
- Power BI
- DAX

---

## Data Preparation & Cleaning
The dataset required extensive preprocessing before analysis:
- Standardized inconsistent date and time formats
- Converted duration fields into analytical formats
- Handled missing values for unresolved tickets
- Validated SLA compliance logic
- Cleaned categorical inconsistencies using snake_case formatting
- Created calculated KPI fields for dashboard reporting

---

## Key KPIs Analyzed
- Total Tickets
- SLA Violation Percentage
- Average Handling Time
- Average Customer Rating
- Tier 2 Ticket Percentage
- Ticket Volume by Channel
- SLA Violations by Country
- SLA Violations by Topic
- Agent Performance Metrics

---

## Dashboard Pages

### 1. Executive Overview
Provides a high-level operational summary including:
- SLA violation trends
- Customer satisfaction distribution
- Ticket distribution by channel and priority
- Major operational bottlenecks

### 2. Agent Performance Analytics
Focuses on:
- Ticket handling volume per agent
- Average handling time
- Customer rating performance
- SLA violations by agent
- Identification of high-performing agents

### 3. Operational Risk & SLA Insights
Highlights:
- SLA violations across countries and priorities
- Tier 2 support ticket distribution
- Handling time by communication channel
- Operational areas requiring resource optimization

---

## Key Business Insights
- Product setup queries contributed the highest number of SLA violations.
- Email support channel handled the majority of incoming tickets.
- Low-priority tickets accounted for the largest operational workload.
- Certain countries demonstrated consistently higher SLA breach percentages.
- Differences in agent handling efficiency and customer satisfaction were identified.
